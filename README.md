# 🌼 Flower Language 花語心情手帳

> 一個結合「羅素情緒模型（Russell's Circumplex Model of Affect）」與植物花語的療癒系手帳 Web App。透過拉動愉悅度與能量滑桿，系統會為用戶推薦專屬的今日代表花卉，並提供手帳客製化、匿名社群廣場、虛擬商店與管理員後台等完整功能。

---

## 🌸 核心特色

*   **🌈 心情座標（羅素情緒模型）**：結合心理學理論，讓用戶拉動「愉悅度 (Valence)」與「能量水平 (Arousal)」滑桿，量化當下的真實感受，並以此推薦最貼近心情的花卉。
*   **📔 療癒手帳日記**：支援純文字撰寫、照片上傳预览、擁有的貼紙裝飾以及手帳背景即時切換。
*   **🌸 動態花語圖鑑**：完整記錄所有花卉的心理特徵座標（V / A 值）與其背後溫暖的療癒花語，記錄完心情後即可解鎖收藏。
*   **🛍️ 虛擬商店與客製化**：用戶可透過每日記錄獲得的心情點數（預設贈送 500 點），自由兌換精心設計的漸層背景與可愛的手繪裝飾貼紙。
*   **💬 匿名社群廣場**：開啟匿名分享開關後，日記將以花朵化名（如：*溫柔的薰衣草 #321*）投遞至廣場，社群成員間可透過「給予擁抱 🫂」與「送一朵花 🌷」進行無壓力的溫暖互動。
*   **🛠 一站式管理後台**：針對管理員權限專門設計，提供即時「花卉上架/刪除」與「商品上架/刪除」功能，資料全自動同步至全站。

---

## 🛠 技術棧

*   **前端框架**：[Petite-Vue](https://github.com/vuejs/petite-vue) (輕量化 Vue 漸進式增強驅動，極速加載)
*   **後端雲服務 (BaaS)**：[Firebase v12+](https://firebase.google.com/)
    *   **Authentication**：用戶登入與權限分流控管
    *   **Cloud Firestore**：即時分佈式 NoSQL 資料庫（用於儲存使用者、日記、花卉圖鑑、商店商品及社群貼文）
*   **UI/視覺設計**：響應式 CSS3 (完美適配手機與桌機)、溫暖的柔和漸層配色、毛玻璃導覽列 (Backdrop-filter)、流暢的微動畫效果 (彈出與淡入)。

## 📂 資料庫結構規劃 (Cloud Firestore)

本專案採用的 NoSQL 資料架構如下：

### 1. `flowers` 集合 (花卉圖鑑)
```json
{
  "_id": "f_sunflower",
  "name": "向日葵",
  "word": "信念、光輝與耀眼的愛",
  "v": 80,
  "a": 80,
  "img": "🌻",
  "type": "flower"
}
```

### 2. `items` 集合 (虛擬商店商品)
```json
{
  "_id": "bg_sunset",
  "name": "晨曦落葉漸層",
  "type": "bg", // bg 或 sticker
  "price": 200,
  "val": "linear-gradient(135deg, #FFEDD5, #FED7AA)" // CSS 樣式或貼紙 Emoji
}
```

### 3. `users` 集合 -> `diaries` 子集合 (用戶專屬手帳)
```json
{
  "_id": "d_1718131200000",
  "type": "diary",
  "username": "user123",
  "text": "今天完成了重要專案，心情真好！",
  "date": "2026年6月12日 晚上11:00",
  "flower": { "id": "f_sunflower", "name": "向日葵", "word": "...", "img": "🌻" },
  "photo": "data:image/png;base64,...", // 支援本地 FileReader Base64 預覽
  "stickers": ["🌸", "✨"],
  "isAnonymous": true
}
```

### 4. `communityPosts` 集合 (社群廣場貼文)
```json
{
  "_id": "p_1718131200000",
  "type": "community-post",
  "username": "user123",
  "nickname": "勇敢的向日葵 #582",
  "text": "今天完成了重要專案，心情真好！",
  "photo": "data:image/png;base64,...",
  "flower": { "id": "f_sunflower", "name": "向日葵", "img": "🌻" },
  "date": "2026年6月12日 晚上11:00",
  "hugs": 5,
  "flowersGiven": 3
}
```

## 🚀 快速開始

1.  **複製程式碼**：將 `index.html` 的完整程式碼複製到本地。
2.  **配置 Firebase**：專案已貼心內置了專屬的 Firebase 開發用金鑰配置，開箱即可自動初始化預設的花卉與商品資料。
3.  **運行應用**：使用 VS Code 的 `Live Server` 擴充功能，或直接雙擊 `index.html` 即可在瀏覽器中執行！
4.  **體驗分流**：
    *   **一般用戶體驗**：在登入頁面選擇 `🌿 一般用戶` 權限，輸入任意帳號密碼註冊或登入，即可開始寫日記、逛商店、去廣場互動。
    *   **管理員體驗**：在登入頁面選擇 `🛠 管理員` 權限登入，右下角會自動解鎖「管理後台」標籤頁，能即時為全站上架全新的神祕小花與限定背景。

## 🎨 介面美學設計

*   **紙質基調**：背景使用 `#FDFBF7` 搭配超細緻的 SVG 噪點濾鏡，模擬手帳紙張的觸感。
*   **古典與手寫字體**：引入 Google Fonts 的 `Noto Serif TC` 古典襯線體與 `Long Cang` 奔放手寫體，打造極具溫度的排版。
*   **自然意象色調**：採用燕麥奶茶色 (`#F5EBE6`)、夕陽暖橘 (`#E6A15C`)、復古墨水黑 (`#5A4632`) 與森林葉綠 (`#8FA06A`)，視覺極致舒適不疲勞。
